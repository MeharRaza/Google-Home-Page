# Google Homepage Clone

A simple and responsive Google Homepage Clone created using HTML and CSS. This project is designed for beginners to practice frontend web development and improve UI design skills by recreating the layout of Google's homepage.

---

## 🚀 Features

- Clean Google-inspired UI
- Responsive layout
- Search bar design
- Navigation links
- Footer section
- Beginner-friendly code structure

---

## 🛠️ Technologies Used

- HTML5
- CSS3
